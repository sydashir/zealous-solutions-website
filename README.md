## Setup Instructions

1. **Download/Clone**: All files should be in the same directory
2. **Images**: Replace placeholder image paths with actual images:
   - `hero-image.jpg` - Hero section image (call centre/office)
   - `about-image.jpg` - About section image (team/office)
   - `client1.jpg` - Testimonial client photo
   - `client2.jpg` - Testimonial client photo
3. **Logo**: Use the provided `logo.svg` or replace with your actual logo
4. **Contact Info**: Update contact information in both HTML and any forms
5. **Content**: Customize text content to match your specific services

## Customization

### Colors
The website uses CSS custom properties for easy color customization:
```css
:root {
    --primary-color: #E67E00;    /* Orange/Golden */
    --secondary-color: #1E3A8A;  /* Dark Blue/Navy */
    --accent-color: #F59E0B;     /* Lighter Orange */
}
```

### Services
Modify the services in the HTML file under the "Services Section". Add or remove service cards as needed.

### Industries
Update the industries section to match your target markets.

### Testimonials
Replace placeholder testimonials with real customer feedback.
